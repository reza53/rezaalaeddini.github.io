Bayesian Approach to DNA Mixture Interpretation

Quantifying evidential strength with probabilistic genotyping

The Likelihood Ratio Framework

Forensic DNA interpretation relies on Bayes’ Theorem:

\[LR = \frac{P(E|H_p)}{P(E|H_d)}\]

Probabilistic Genotyping

Modern systems like STRmix™ model allele peak heights as:

\[y_i \sim \text{Gamma}(a \cdot x_i, b)\] # Likelihood Ratio (LR) Example for Forensic DNA # Assume the prosecution model gives 90% probability # and the defense model gives 30% probability prosecution_prob = 0.90 defense_prob = 0.30 # Calculate the LR lr = prosecution_prob / defense_prob # Print the result print("Likelihood Ratio:", lr) Thresholds for LR interpretation:
  • LR > 1,000: Strong support for Hp
  • LR < 0.001: Strong support for Hd

Challenges

1. Stochastic Thresholds

Low-template DNA requires:

\[P(y_i > 0 \mid x_i = 0) = 1 - e^{-\lambda}\]

2. Relatedness

Adjusting for kinship:

\[LR_{\text{related}} = \frac{1 + \theta}{(1-\theta) + \theta \cdot F}\]
